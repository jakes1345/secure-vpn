General Questions

What is PhazeVPN?

PhazeVPN is a military-grade VPN service that encrypts your internet connection and protects your privacy. We use ChaCha20-Poly1305 encryption (stronger than AES-256) and enforce a strict no-logs policy.

How does PhazeVPN protect my privacy?

PhazeVPN uses network-level security features including kill switch, DNS leak protection, IPv6 routing, and WebRTC protection. All security is enforced at the VPN level - no browser extensions needed.

Do you keep logs?

No. We have a strict no-logs policy. We don't track your browsing activity, connection times, or IP addresses. Your privacy is our top priority.

Installation & Setup

How do I install PhazeVPN?

1. Download the client from our download page
2. Install Python 3 and OpenVPN on your system
3. Run the client and login with your account
4. Click Connect and you're protected!

What operating systems are supported?

We support Windows 10/11, Linux (Ubuntu/Debian), and macOS 10.14+. Mobile users can use OpenVPN Connect app with our .ovpn config files.

Do I need to install anything else?

Yes, you need Python 3 and OpenVPN installed. See our setup guide for detailed instructions.

Features & Security

What is a kill switch?

A kill switch blocks all internet traffic if your VPN connection drops. This prevents your real IP address from being exposed. PhazeVPN's kill switch is enforced at the network level, making it impossible to bypass.

What encryption does PhazeVPN use?

We use ChaCha20-Poly1305 encryption, which is cryptographically equivalent to AES-256-GCM but often faster. We also use 4096-bit RSA certificates and TLS 1.3 for maximum security.

Can I use PhazeVPN for streaming?

Yes! PhazeVPN is optimized for speed and can handle streaming, gaming, and video calls without lag.

Troubleshooting

I can't connect to the VPN

Check that OpenVPN is installed and running. Make sure port 1194 (UDP) is not blocked by your firewall. See our setup guide for more help.

My connection is slow

Try connecting to a different server location. Check your internet speed. If issues persist, contact support.

I forgot my password

Use the Forgot Password link on the login page to reset it.

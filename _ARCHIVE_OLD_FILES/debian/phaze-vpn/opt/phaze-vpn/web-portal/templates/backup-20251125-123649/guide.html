üöÄ Quick Start
  1. Create an account - Sign up for free at the signup page
  2. Login - Access your dashboard
  3. Create a VPN client - Click "Create New Client" in your dashboard
  4. Download OpenVPN Connect - See instructions below for your device
  5. Import your config - Download your .ovpn file or scan the QR code
  6. Connect! - Tap/click connect and you're protected!

üíª Windows Setup

Step 1: Download OpenVPN

  1. Visit openvpn.net/community-downloads
  2. Download OpenVPN for Windows
  3. Install it (keep default settings)

Step 2: Import Your Config

  1. In your dashboard, click ‚¨áÔ∏è Download for your client
  2. Save the .ovpn file
  3. Right-click the OpenVPN icon in your system tray (bottom right)
  4. Click "Import file..."
  5. Select your downloaded .ovpn file

Step 3: Connect

  1. Right-click the OpenVPN icon in system tray
  2. Select your VPN profile
  3. Click "Connect"
  4. ‚úÖ You're protected!
üçé macOS Setup

Step 1: Download Tunnelblick

  1. Visit tunnelblick.net
  2. Download and install Tunnelblick (free, open-source)
  3. Or use OpenVPN Connect from the Mac App Store

Step 2: Import Your Config

  1. Download your .ovpn file from the dashboard
  2. Double-click the .ovpn file
  3. Tunnelblick/OpenVPN will automatically import it

Step 3: Connect

  1. Click the Tunnelblick/OpenVPN icon in your menu bar (top right)
  2. Select your VPN profile
  3. Click "Connect"
  4. ‚úÖ Connected!
üêß Linux Setup

Method 1: Command Line (Easiest)

  1. Install OpenVPN: sudo apt install openvpn
  2. Download your .ovpn file from the dashboard
  3. Connect: sudo openvpn --config your-client.ovpn

Method 2: Network Manager GUI

  1. Install: sudo apt install network-manager-openvpn
  2. Download your .ovpn file
  3. Open Settings ‚Üí Network
  4. Click "+" ‚Üí "Import from file..."
  5. Select your .ovpn file
  6. Toggle the VPN connection in network settings
üí° Pro Tips
  • Each client can only connect once at a time - create multiple clients for multiple devices
  • QR codes make setup super easy on mobile - just scan and connect!
  • Keep your .ovpn files secure - they're like passwords
  • You can download and reconnect anytime from your dashboard
  • If you have issues, check that the VPN server is running (shown on dashboard)
